# Cubey World

3D open-world driving game built with Three.js. Single HTML file, no install needed — just open in a browser.

## Features

- Massive 4000x4000 open world with rolling hills (Perlin noise terrain)
- Roads that follow terrain height: highways, city streets, and dirt trails
- City center with grid layout, 5 outlying villages connected by highway ring
- Car with acceleration, braking, steering, and terrain-following physics
- Mouse drag to orbit camera, scroll to zoom
- Minimap with road network

## Controls

- W / Arrow Up — Accelerate
- S / Arrow Down — Brake / Reverse
- A / Arrow Left — Steer left
- D / Arrow Right — Steer right
- Space — Handbrake
- R — Reset car
- Mouse drag — Orbit camera
- Scroll — Zoom

## Road Types

- Highway (grey, 8m wide, yellow center lines) — full speed
- City street (dark grey, 7m wide, white lines) — slightly slower
- Dirt trail (brown, 3.5m wide, no lines) — capped at 40 km/h
